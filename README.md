## Description

Full-stack application that will allow the user to register, write one paragraph of motivational quote, review already registered users and their quotes, and upvote or downvote a quote.

- CSS

## How To Use

#### Installation

Clone this repository and run:

```bash
# install dependencies
$ yarn install
```

#### Running the app

```bash
# development
$ yarn run start
```

```bash
# watch mode
$ yarn run start:dev
```

```bash
# production mode
$ yarn run start:prod
```

#### Test

```bash
# unit tests
$ yarn run test
```
```bash
# watch mode
$ yarn run test:watch
```
```bash
# e2e tests
$ yarn run test:e2e
```
```bash
# e2e tests - watch mode
$ yarn run test:e2e:watch
```
```bash
# test coverage
$ yarn run test:cov
```
